EEG Signal Processing & Brain Interfaces

Real-time EEG Artifact Detection

This video demonstrates real-time detection of various EEG artifact classes, which will later be used as hand movement switches for rehabilitation purposes. Advanced signal processing techniques are applied for accurate classification.

Demonstration of EMG signal acquisition from a patient during experimental sessions. This shows the practical application of biomedical signal processing in clinical environments.

Hand Gesture Recognition & Computer Vision

Hand Gesture Recognition

Hand gesture recognition implementation using Python with MediaPipe and QtPy libraries. This demonstrates real-time computer vision applications for human-computer interaction.

5 Finger Angle Plot

Real-time finger angle calculation and plotting using Python with MediaPipe and QtPy. The system detects hand landmarks, calculates finger angles, and visualizes them dynamically.

FES Systems & Neurorehabilitation

FES Hand Stimulation: Paragrasp Glove Test (Healthy Subject)

Investigation of the Paragrasp functional electrical stimulation (FES) glove effects on a healthy subject. This advanced neurorehabilitation device features 8 reference points and 32 stimulation channels.

FES Hand Stimulation: Paragrasp Glove Test (Tetraplegic Patient)

Investigation of the Paragrasp functional electrical stimulation (FES) glove effects on a tetraplegic patient with cervical spinal cord injury. Demonstrates real clinical applications of FES technology.

Patient Walking with ParaWalk

A spinal cord injury patient walking with the aid of ParaWalk, a neuroprosthetic device designed to help individuals with spinal cord injuries stand and walk. Shows the practical impact of assistive technologies.
